# Ruff Driver
这是一个4路驱动板。 
针脚A B C D 分别连接电机正负极， + - 连接直流电正负极
每个针脚分别决定电机的转动方向，如果方向有误，请调换电机的正负极。
仅用于RUFF开发板
##接口说明

小车前进

```js
$("#hg7881").turnUp();
```

小车后退

```js
$("#hg7881").turnDown();
```

小车左转

```js
$("#hg7881").turnLeft();
```

小车左转

```js
$("#hg7881").turnRight();
```
